h1. Dutch Language Pack for ExpressionEngine 2.x

Originally by "Willem de Boer":http://www.willemdeboer.com/, added to GitHub for convenience.

h2. Installation

* "Download":https://github.com/lodewijk/dutch.lang/zipball/master the latest version
* Rename the folder you downloaded to *dutch*
* Move the @dutch@ folder to your @/system/expressionengine/language/@ directory

h2. Contains

* Core EE language files
* Discussion Forums language files
* MSM expansion language files
